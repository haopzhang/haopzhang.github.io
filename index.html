Research Interests

Artificial intelligence technologies, e.g. image processing, pattern recognition, computer vision, machine learning, etc., and their applications in aerospace exploration, remote sensing, and medical image analysis.

Special interests recently include

  • Deep learning
  • Image processing and analysis for remote sensing
  • Multi-view object detection and recognition
  • 3D object recognition and pose estimation
  • 3D reconstruction from 2D images
  • ...
